# 3D Animated Portfolio Website

A **futuristic 3D animated personal portfolio website** showcasing my journey into modern web development and interactive 3D experiences.  
This project blends **clean UI design** with **real-time 3D visuals** to create an immersive first impression.

## 🚀 Features

- 🎮 Real-time 3D animated model
- 🌌 Dark futuristic UI design
- 🖱️ Interactive visual elements
- ⚡ Smooth animations and transitions
- 📱 Responsive layout (desktop-first)
- 🔗 Social contact integration (Instagram & WhatsApp)

## 🧠 About the Project

This portfolio represents my learning and exploration of:

- Modern front-end development
- 3D web experiences
- Interactive UI/UX design
- Performance-aware animation usage

The goal was to create something **more than a static portfolio** — a digital presence that feels alive.
.
## 🛠️ Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **Spline (3D design & animation)**

> No frameworks used — keeping it lightweight and beginner-friendly.
